SatoshiLabsHomework
===================

### Prerequisites

[Node.js](http://nodejs.org) is required.
```
npm install -g gulp
npm install -g bower
```

### Create App

```
git clone https://github.com/Hurtak/SatoshiLabsHomework.git
cd SatoshiLabsHomework
npm install
bower install
```

### Usage

minify js and css, prefix css, run from `dist` folder

```
gulp
```

run from `app` folder

```
gulp dev
```
